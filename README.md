# MediCare AI 🩺💊

MediCare AI is a smart health assistant web application that helps users manage daily health tasks and get basic AI-based health guidance.

## 🌟 Features
- 💊 Medicine Reminder System  
- 📅 Doctor Appointment Scheduler  
- 🩺 AI-Based Symptom Checker (Rule-based Intelligence)  
- 👨‍👩‍👧 Family Medicine Tracker  
- 📞 Emergency Contact Storage  
- 💧 Water Intake Tracker  
- 🌙 Dark Mode Support  
- 📊 Health Dashboard with Local Storage  
- 💡 Daily Health Tips Generator  

## 🧠 Microsoft IQ Integration (Hackathon Requirement)
This project includes a **simulated integration of Microsoft Work IQ / Foundry IQ concepts**, where:

- The Symptom Checker acts as an AI-powered decision engine
- Health suggestions are generated using intelligent rule-based logic
- Data insights and tracking simulate Fabric IQ-style analytics

> Note: This is a frontend implementation demonstrating AI/IQ-inspired behavior using JavaScript logic.
